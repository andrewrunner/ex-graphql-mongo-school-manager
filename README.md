
Simple backend api for school mamanger app;

### Use
- [NestJS](https://nestjs.com/)
- MongoDB + TypeORM
- GraphQL (graphql-tools, apollo)
- jest
- class-validator - DTO class fields validation

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# Open schema development mode
# http://localhost:3000/graphql 

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
